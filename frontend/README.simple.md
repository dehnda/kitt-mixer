# K.I.T.T. Cocktail Mixer Frontend

React TypeScript frontend with Knight Rider K.I.T.T. theme, optimized for 480x320 touchscreens.

## Quick Start

```bash
npm install
npm start  # Development mode at http://localhost:3000
npm run build  # Production build
```
